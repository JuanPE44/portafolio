# Mi Portafolio

Este es mi portafolio personal, desarrollado utilizando React y Styled Components. El portafolio muestra mis proyectos, habilidades.

## Demostración

![portafolio](https://github.com/JuanPE44/portafolio/assets/89142353/744c62ff-df78-4f02-bcc9-0b43cf200885)


## Características

- Diseño moderno y responsive que se adapta a diferentes tamaños de pantalla.
- Sección de proyectos destacados que muestra una descripción breve de cada proyecto.
- Sección de habilidades que destaca mis conocimientos técnicos y competencias relevantes.
- Enlaces a mis perfiles de redes sociales y contacto para facilitar la comunicación.

## Instalación

1. Clona este repositorio en tu máquina local: `git clone https://github.com/JuanPE44/portafolio.git`
2. Navega hasta el directorio del proyecto: `cd portafolio`
3. Instala las dependencias: `npm install`
4. Inicia la aplicación: `npm run dev`
